
## 项目代码结构

```
BiliCrawler
├── base 
│   └── base_crawler.py         # 项目的抽象类
├── config 
│   ├── base.py                 # 基础配置
│   └── credential.py           # 登录参数配置
├── examples                    # 示例目录  
├── modules
│   ├── colu_section            # 专栏分区crawler实现
│   ├── emoji                   # 表情包crawler实现
│   ├── ip_address              # ip终端信息crawler实现  
│   |── music                   # 音频crawler实现
│   |── search                  # 搜索crawler实现
│   |── topic                   # 话题crawler实现
│   |── user                    # 用户crawler实现
│   |── video                   # 视频crawler实现
|   └──
├── results                     # 数据保存目录  
├── test                        # 测试案例目录  
├── tools
│   └── utils.py                # 暴露给外部的工具函数
└── main.py                     # 程序入口
```
